# COS 106 Student Portfolio Website

This folder contains a complete static multi-page website for the COS 106 term project.

## Pages

- `index.html` - homepage with student name, photograph, biography, and navigation
- `about.html` - education table, career aspiration, skills, hobbies, and audio multimedia
- `projects.html` - three sample projects with images, descriptions, and links
- `planner.html` - interactive task planner using JavaScript
- `contact.html` - contact form with JavaScript validation

## How to Use

Open `index.html` in a browser to view the website. Edit the sample name, biography, email, phone number, and project details before submitting.

## Project Requirements Covered

- Semantic HTML pages
- External CSS stylesheet
- Responsive design with Flexbox, CSS Grid, and media queries
- Images, hyperlinks, lists, forms, tables, and audio
- JavaScript event handling, DOM updates, arrays, functions, task management, and contact form validation
